Installation on Ubuntu

1. Clone GIT repo of Sakai (https://github.com/sakaiproject/sakai.git) into ./Sakai sub-directory
   of this directory (the directory of the README.txt file).
2. Check out the commit: 96ef4602e640872c37e7f9b09d5f3a28d22c2ab9
3. Open terminal in this directory and run there this command: python ./build.py
   Note: the Python script is already present in this directory.

Resulting WAR/JAR files are copied into directory ../BENCHMARK, which is a destination
already consistent with this evaluation.
