Encuestame
~~~~~~~~~~

Encuestame is an Open Source Java web application to publish, collect, share and
measure the opinion of the people thought the social media or private
environments. Private or public surveys, protected by password, widgets
embeddable, ready for mobile and multi language.

1. Open a terminal in the directory of this readme file and clone encuestame:
      git clone https://github.com/encuestame/encuestame
2. Rename the created directory "encuestame" to "Encuestame"
3. Enter the directory and build with maven:
      cd Encuestame/
      git checkout 42cb745cbf15c9b40e497242898c2f75ded278f7 .
      mvn clean package
4. Copy the resulting files to ../BENCHMARK (relative path to this README.txt
   file):
      cd .. 
      python ./copy_binaries.py
