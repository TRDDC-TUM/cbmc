Restaurant Chooser
~~~~~~~~~~~~~~~~~~

Small application implementing a voting system for deciding where to have lunch.
2 commits
1 branch
0 releases
0 contributors

Repository: https://github.com/michaelzherdev/restaurant_chooser


1. Open a terminal in the directory of this readme file and clone:
git clone https://github.com/michaelzherdev/restaurant_chooser
2. Enter the directory and build with maven:
cd restaurant_chooser/
mvn clean package

3. Copy the resulting files to ../BENCHMARK (relative path to this README.txt file):
mkdir ../../BENCHMARK
mv target/* ../../BENCHMARK/
