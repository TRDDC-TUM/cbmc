Restaurant Chooser
~~~~~~~~~~~~~~~~~~

Small application implementing a voting system for deciding where to have lunch.
2 commits

0 contributors

Repository: https://github.com/michaelzherdev/restaurant_chooser


1. Open a terminal in the directory of this readme file and clone:
      git clone https://github.com/michaelzherdev/restaurant_chooser
2. Enter the directory and checkout the right commit:
      cd restaurant_chooser/
      git checkout df869799605fcd820ca1b9de6f821584b20d1fe6 .
3. And build the application with maven:
      mvn clean package
4. Copy the resulting files to ../BENCHMARK (relative path to this README.txt
   file):
      cd .. 
      python ./copy_binaries.py
